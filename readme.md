キャラクターのセリフを表形式でまとめるアプリを作ります。
アプリはサーバを起動してブラウザで開ける形にしたいです。
一度ブラウザを閉じても再び起動した時、以前の状態を復元できるようにしてください
・ブラウザ起動時はすでに登録したキャラクターの表を起動するか、キャラクター専用の表を新規作成するかのボタンを表示する
・すでに登録したキャラクターの表を起動する場合は、ポップアップなどから一覧を表示して具体的にどの表を開くのか選択する
・表は、列は「セリフ」「ストーリー名」の２列にしてください
・行は行追加ボタンから増やせるようにしてください
・また、行削除ボタンも追加してください
・行削除ボタンを押すと、「削除する行を選択してください」と表示し、ボタンを押した場合にのみその後行をクリックすると青く光って選択状態にできるようにしてください
・削除するボタンを再び押すと、選択した行は削除するようにしてください

ホーム画面で、新たに表を削除ボタンを追加してください
削除ボタンを押すと、既存の表を開くと同様にキャラ一覧を表示してください
名前をクリックすると青く光り選択状態にし、削除するというボタンをあらためてクリックすると削除できるようにしてください

・表を開いた後の画面で、入力したセリフやストーリー名を検索で絞り込みできるようにしてください
・また、表を新規作成するときに表の名前を設定する他にタグも追加で設定できるようにしてください

・ホーム画面の背景が黒いので、元の薄いピンクとピンクの水玉模様に戻してください
・新規作成するときの「表の名前を記入してください」というポップアップと一緒にタグの設定ができるようにしてください
・また、既存の表を開くを押したときに出るポップアップでタグによる絞り込みができるようにしてください
・既存の表を開いた後の「タグで絞り込み」という表示は要りません
・既存の表を開いた後、タイトル(キャラクター名が表示されている部分)の隣にタグを表示するようにしてください

・背景は水玉模様にしてください
・表には現在セリフとストーリー名がありますが、左端などに透明のハートマークを置いてください(お気に入り登録機能)
・透明のハートを押すと赤く色づき、お気に入り登録ができるようにしてください
・表がみれる画面で、お気に入り登録した行で絞り込めるようにもしてください

・表を表示した際に左端がお気に入り登録に関する列となっていますが、お気に入り登録マークは列として表示しないでください。
・列としては無くなるので「お気に入り」という列名も表示しないでください。
・最終的に表示されるのはハートマークのみでお願いします
・背景を水玉に変更してもらいましたが、全ての遷移画面でそのような背景にしてください

・表を表示したときに、セリフやストーリ名の列と行のそれぞれの項目と縦が一致していないので整えてください
・また左端にあるお気に入り登録マークが、各行に対応して表示されていないので各行で表示できるようにしてください

・表で記入したセリフを参考にAIが学習できるようにしてください
・ホーム画面で「セリフ分析」という項目を追加してください
・セリフ分析を押すと、ユーザが独自に自由にセリフを入力できる画面に遷移してください
・入力した独自のセリフが学習したセリフの特徴とどれくらい一致しているか分析する機能を実装してください
・このとき、どの表のデータから分析するか選択するボタンも表示してください

・表にあるセルにカーソルを合わせたときだけ、対象のセルの全文を表示するような仕様にしてください
・できれば吹ふき出しのフレームで表示して欲しいです

・セルにカーソルを合わせても変化がありません。全文が小さく表示されるようにしてください。
・また、文字変換をする際に文字が白くて見えにくいです。文字変換のフォントの色はピンクにしてください

・ホーム画面の「既存の表を開く」は「セリフを見る」、「表を削除」は「データを削除」に名称を変更してください。
・新規作成を押すと
「表の名前を入力してください:

（タグを設定する場合は、名前の後にカンマで区切って入力してください）
例: メインキャラ,主人公,ヒロイン」
と出ますが、これを
「キャラクター名を入力してください:

（タグを設定する場合は、名前の後にカンマで区切って入力してください）
例: 〜〜(名称),主人公,ヒロイン
に直してください

・表を開いた後に上部にホームに戻るとありますが、その左隣に「タグを編集」ボタンを作成してください
・「タグを編集」を押すと、設定しているタグを編集できるようにしてください

・ホーム画面で「テーマ設定」ボタンを追加してください
・テーマ設定ボタンを押すと、カラーリングを変更するためのポップアップが表示されます
・「いちごみるく」「コットン」「プリンアラモード」いずれかを選択できるようにしてください
・いちごみるくでは、現在なっているような色に変更します
・コットンでは「紫、ピンク、白、青」を基調としたカラーに変更してください
・プリンアラモードでは「茶色、黄色、白、赤」を基調としたカラーに変更できるようにしてください

・セリフを見るで遷移した画面の背景は、ホームと同じ水玉模様にしてください。
・「いちごみるく」「コットン」「プリンアラモード」それぞれ選択したときは、それぞれのカラーリングに合うような水玉の背景になるようにしてください。
・セリフを見るで遷移した画面でも水玉の背景にして、カラーリングだけ変えるようにしてください

・コットンは色味をもっと薄くパステルにしてください。
・セリフを見るで遷移した画面で、「行を削除」「保存」はカラーリング設定を変えても同じなので変更します。「いちごみるく」は今のまま、「コットン」はそれぞれ青かピンク、「プリンアラモード」は黄色と赤にしましょう

## 起動方法

このアプリはNode.js + Expressで動作するWebアプリケーションです。以下の手順で起動できます。

### 前提条件
- Node.js (バージョン14以上推奨) がインストールされていること

### 手順
1. リポジトリをクローンまたはダウンロードし、プロジェクトディレクトリに移動します。
2. 依存パッケージをインストールします。
   ```bash
   npm install
   ```
3. サーバーを起動します。
   ```bash
   npm start
   ```
   または
   ```bash
   node server.js
   ```
4. ブラウザで以下のURLを開きます。
   ```
   http://localhost:3000
   ```
5. アプリケーションが表示され、キャラクターセリフ管理が利用できます。

### 環境変数
- ポート番号を変更する場合は、環境変数`PORT`を設定してください。
  例:
  ```bash
  PORT=8080 npm start
  ```

### 開発モード
- 開発時も同じコマンドで起動します（ホットリロードは未設定）。
